# Backend Arch Pro Max Release Notes
 

## Version 0.2.0
 
- Release: `0.2.0`
- Date: `2026-05-04`
- Tag: `v0.2.0`
 
### Highlights
 
- **Rich Metadata & Decision Support**: Every architectural rule now includes performance tiers (throughput, latency) and cost/complexity ratings to support better programmatic decision making.
- **Trust & Credibility**: Added `source_url` and `last_updated` to all records, allowing agents to verify information against official docs and engineering blogs.
- **New 'Compare' Command**: Easily generate trade-off tables between competing technologies or patterns.
- **Improved Reliability**: Search engine refactored for better domain accuracy and result confidence scoring.
- **Maintenance Tools**: Built-in stale-checking to ensure the knowledge base stays fresh with current industry standards.
